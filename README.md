# Python_program
A collection of Python scripts and projects showcasing fundamental programming concepts, problem‑solving techniques, and practical applications. This repository is designed for learners, interview preparation, and anyone exploring Python’s versatility.

Basic Programs – Simple scripts to understand syntax and logic

Data Structures – Implementations of lists, stacks, queues, linked lists, trees, and graphs

Algorithms – Sorting, searching, recursion, and dynamic programming examples

File Handling – Reading/writing files, CSV/JSON operations

OOP in Python – Classes, inheritance, polymorphism, encapsulation

Problem Solving – LeetCode/HackerRank style coding challenges
